# Mustafa Shaikh | Frontend Developer Portfolio

A premium, interactive personal portfolio built with a focus on "Snow Mountain" aesthetics—crisp, minimal, and high-performance.

**Live Site:** [https://mustafashaikh3060-lgtm.github.io/portfolio](https://mustafashaikh3060-lgtm.github.io/portfolio)

## 🏔️ The Vision
Inspired by the snowy peaks of Manali, this portfolio blends sharp, professional typography with fluid, physics-based interactions. It reflects a commitment to clean code and elegant user experiences.

## 🚀 Technical Highlights
- **Engineered with TypeScript:** Robust, type-safe frontend architecture.
- **Custom Physics Engine:** A refined, canvas-based "Dirty Snow" particle system.
- **Interactive UX:** 
  - Magnetic navigation and contact elements.
  - 3D perspective-tilt imagery.
  - Custom dual-phase minimalist cursor.
- **Automated CI/CD:** Fully integrated with GitHub Actions for seamless deployment.

## 🛠️ Tech Stack
- **Languages:** TypeScript (ES6+), CSS3, HTML5
- **Design:** Editorial Typography (Playfair Display & Inter), Asymmetric Fluid Grids
- **Build & CI/CD:** npm, tsc, GitHub Actions
